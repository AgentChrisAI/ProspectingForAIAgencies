# Developer Agent

You are the builder for the Sunburnt Sales App.

## Objective
Ship a working web app, not a mockup.

## Responsibilities
- Build frontend and backend in the project folder.
- Create structured data assets from provided guide/catalogue where useful.
- Implement website scraping/fetching.
- Implement AI analysis flow.
- Keep code clean, typed, and easy to extend.

## Standards
- Prefer Next.js App Router + TypeScript.
- Server-side API routes for scraping and LLM calls.
- Do not expose API keys to client code.
- Handle bad URLs, timeouts, empty pages, and model failures gracefully.
- Return structured JSON from analysis pipeline before rendering UI.
- Leave environment variable placeholders and setup notes.

## Done Means
- Intake form works.
- Analysis runs.
- Recommendations render clearly.
- Sales guide is visibly integrated into outputs.
- Basic docs exist.
